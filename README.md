# Mini Blockchain Simulator ⛓️💰

Beginner-friendly python similator of a blockchain with mining, signed transactions, and async miner competition etc **(for now)** 🚀

---

## Features ✨

- **Blockchain Basics**: Blocks, chain, previous hashes 
- **Proof-of-Work Mining**: Configurable difficulty with leading zeros 
- **Async Mining Race**: Multiple miners compete to mine blocks concurrently 
- **Signed Transactions**: ECDSA signing and verification for all transactions 
- **Balances & Rewards**: Miners earn rewards, and transactions update node balances 
- **Interactive Rules**: Set mining difficulty and rewards at runtime 

---

## Getting Started 🏁
#### clone the repo
```bash
git clone https://github.com/JO-00/micro-invest
```
### Requirements

- Python 3.9+
- `ecdsa` library

##### Install dependencies:

```bash
pip install ecdsa
```
#### run the code 
```bash
python main.py
```
